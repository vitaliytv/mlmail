## Огляд

Надає `Endpoints` для читання налаштованих адрес доступу. Не змінює файли чи дані в базі.

## Поведінка

1. Endpoints формує стандартний набір адрес для взаємодії з Google та Gmail.
2. Набір охоплює обмін токеном, роботу з вхідною міткою, мітками, повідомленнями, пакетною зміною та фільтрами.
3. Значення адрес визначаються централізованими налаштуваннями відповідних інтеграцій, щоб усі операції використовували узгоджені точки доступу.
4. Створення Endpoints не змінює файли чи дані в базі.
