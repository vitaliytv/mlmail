Діалог показує журнал запитів агента з можливістю розгортати окремі записи, відповідати на уточнення, підтверджувати або відхиляти очікуючі дії та запускати аналіз запису. Він працює як локальне розширення журналу, додаючи до кожного рядка панель аналізу без зміни загального вигляду списку.

## Поведінка

1. Відкриває діалог із журналом запитів і відображає список записів, отриманих від агента.
2. Під час відкриття діалогу завантажує оточення для локального аналізу та оновлює список записів.
3. Якщо записів немає, показує стан порожнього журналу.
4. Клік по рядку запису розгортає або згортає його деталі.
5. У розгорнутому записі показує відповідь на запит, а для записів з очікуванням підтвердження — окремий блок погодження дії.
6. Дозволяє підтвердити або відхилити небезпечну дію, після чого оновлює журнал і повідомляє про зміну назовні.
7. Дозволяє надіслати текстову відповідь на уточнення, після чого також оновлює журнал і повідомляє про зміну назовні.
8. Для кожного розгорнутого запису надає запуск аналізу через cloud-режим або локальний режим.
9. Показує окремо стан помилки та результат аналізу для кожного запису.
10. Під час довгих операцій блокує повторні дії для відповідного запису та показує індикатор завантаження.

## Гарантії поведінки

- Журнал завжди оновлюється після відповіді, підтвердження або відхилення.
- Помилки операцій не ламають діалог, а показуються користувачу як повідомлення.
- Аналіз виконується незалежно для кожного запису й не змішує результати між ними.
- Локальний аналіз доступний одразу після відкриття діалогу.
- Порожній стан відображається лише тоді, коли список справді порожній і немає активного завантаження.
