## Огляд

Компонент `TemplateEditorFields.vue` надає інтерфейс для редагування полів, що описують шаблон повідомлення. Він дозволяє користувачу вводити назву, обирати тип шаблону (Newsletter або Task) та конфігурувати параметри, залежно від обраного типу.

## Поведінка

1. Компонент приймає об'єкт моделі через `defineModel` як вхідні дані, який обов'язково повинен містити властивість `name`.
2. Він приймає додаткові підказки (`subjectHint` та `promptHint`) для довідки користувачу.
3. Користувач вводить назву шаблону, змінюючи властивість `model.name`.
4. Користувач вибирає тип шаблону, змінюючи властивість `model.type` між значеннями `'newsletter'` та `'task'`.
5. Користувач може вказати шаблонний підрядок для поля відправника (`model.from_pattern`).
6. Користувач вводить шаблонний підрядок для теми листа (`model.subject_pattern`), використовуючи надану підказку `subjectHint`.
7. Якщо вибраний тип шаблону є `'task'`, користувач може вводити назву завдання (`model.task_label`).
8. Якщо вибраний тип шаблону є `'newsletter'`, користувач може вводити інструкцію для LLM (`model.prompt`), використовуючи надану підказку `promptHint`.

## Гарантії поведінки

- Компонент завжди оновлює зовнішню модель (`model`) при зміні будь-якого введеного поля.
- Відображення поля для назви завдання або інструкції для LLM залежить виключно від значення `model.type`.
- Поля для шаблонів (`from_pattern`, `subject_pattern`) є необов'язковими, оскільки вони не мають валідації у компоненті.
