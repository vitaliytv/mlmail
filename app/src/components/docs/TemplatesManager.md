Компонент керує переглядом, створенням, редагуванням і видаленням шаблонів у модальному вікні. Він розрізняє системні та власні шаблони й показує їхній тип, джерело та призначення.

## Поведінка

1. Відкриває список шаблонів лише тоді, коли діалог стає видимим.
2. Показує порожній стан, якщо шаблонів ще немає.
3. Для кожного шаблону відображає:
   - назву;
   - тип;
   - ознаку системного або власного шаблону;
   - умови застосування за відправником і темою;
   - окрему мітку для шаблонів-завдань.
4. Дає змогу створити новий шаблон через окремий редактор.
5. Дає змогу редагувати існуючий шаблон у тому ж редакторі.
6. Автоматично формує ідентифікатор, якщо він ще не заданий.
7. Зберігає шаблон як звичайний або як системний, залежно від режиму.
8. Після збереження системного шаблону прибирає користувацьке перевизначення, щоб новий системний варіант не був прихований.
9. Дає змогу видаляти лише не системні шаблони.
10. Після будь-якої зміни оновлює список шаблонів.
11. У режимі розробки показує додатковий перемикач збереження як системного шаблону.

## Гарантії поведінки

- Системні шаблони не можна видалити з інтерфейсу.
- Кнопка збереження не активується, доки шаблон не має достатніх даних для створення.
- Редактор завжди відкривається з порожнім станом для нового шаблону або з поточними даними для редагування.
- Після успішного збереження або видалення список у вікні залишається актуальним.
- Шаблон-завдання й newsletter-шаблон мають різні обов’язкові поля для збереження.
