## Огляд

Файл містить системні інструкції та промпти для короткого переказу й перекладу email, щоб узгоджувати стиль українського тексту без зайвих службових вставок. Окремі `SUMMARY_SYSTEM`, `TRANSLATE_SYSTEM`, `TRANSLATE_BATCH_SYSTEM`, `buildSummaryPrompt`, `buildTranslatePrompt`, `extractTextNodes`, `translateHtmlEmail` відповідають за підготовку тексту з HTML-листа, виділення видимого змісту та обробку листа так, щоб зберігалися його читабельність, порядок фрагментів і розмітка повідомлення.

## Поведінка

- `SUMMARY_SYSTEM` — задає інструкцію для короткого українського переказу листа: стисло передати суть, відправника, подію та очікування.
- `TRANSLATE_SYSTEM` — задає інструкцію для перекладу людського тексту email українською без службових вставок і збереженням структури.
- `TRANSLATE_BATCH_SYSTEM` — задає інструкцію для пакетного перекладу масиву рядків у JSON-форматі з незмінною кількістю елементів.
- `buildSummaryPrompt` — формує текст запиту для підсумку листа з полями відправника, теми та тіла.
- `buildTranslatePrompt` — формує текст запиту для перекладу листа, прибираючи службовий технічний шум із тіла.
- `extractTextNodes` — збирає всі непорожні текстові вузли з DOM-дерева, пропускаючи службові теги.
- `translateHtmlEmail` — перекладає текстові вузли HTML-листа через пакетний переклад і повертає оновлений HTML.

## Публічний API

- SUMMARY_SYSTEM — системна інструкція для локальної моделі: коротко підсумувати лист українською.
- TRANSLATE_SYSTEM — системна інструкція для локальної моделі: перекласти лист українською, зберігаючи зміст і тон.
- TRANSLATE_BATCH_SYSTEM — системна інструкція для пакетного перекладу: обробляти кілька фрагментів тексту узгоджено.
- buildSummaryPrompt — формує prompt для ролі user, щоб модель підсумувала повідомлення.
- buildTranslatePrompt — прибирає CSS-блоки, які потрапляють у plain-text витяг з HTML-листів, і готує prompt для перекладу.
- extractTextNodes — збирає всі непорожні текстові вузли DOM разом зі стабільними ключами-заповнювачами; окремо повертає вузли й тексти для моделі.
- translateHtmlEmail — розбирає HTML, перекладає всі текстові вузли через `translateBatch` і повертає перекладений HTML.
