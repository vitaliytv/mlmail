Цей компонент є головним екраном застосунку: він поєднує вхід користувача, фонове оновлення клієнта та панель контролю задач. Після авторизації він запускає перевірку шаблонів і показує стан сканування в інтерфейсі.

## Поведінка

1. Ініціалізує стан автентифікації, сканування задач і механізм оновлення застосунку.
2. Показує екран входу для всіх користувачів.
3. Коли користувач стає авторизованим:
   - отримує список шаблонів;
   - запускає сканування цих шаблонів;
   - оновлює стан прогресу сканування.
4. Поки користувач авторизований, відображає панель задач із поточними даними:
   - список задач;
   - статус сканування;
   - кількість уже перевірених елементів;
   - загальну кількість елементів.
5. Коли задача позначена як завершена в панелі, знімає відповідну позначку з повідомлення.

## Гарантії поведінки

- Сканування не запускається для неавторизованого користувача.
- Панель задач не відображається без авторизації.
- Після входу стан задач оновлюється автоматично.
- Користувач бачить лише актуальний прогрес сканування.
- Завершення задачі в UI синхронізується зі станом застосунку.
