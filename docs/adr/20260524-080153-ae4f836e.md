## Що вже побудовано

### Бекенд (Rust + Tauri)

Повноцінна інтеграція з Google:

- **OAuth 2.0 (PKCE-flow)** — логін через системний браузер (macOS) або WebView (Android)
- **Token management** — refresh token на диску (macOS: JSON-файл, Android: JNI secure storage), access token кешується в памʼяті з перевіркою expiry
- **Gmail API** — два запити: кількість листів у скринці та деталі конкретного листа
- **MIME-парсер** — витягує `text/plain`, якщо немає — стрипає HTML, розуміє кодування ISO-8859-1 / Windows-1251 / UTF-8, tolerates base64url без padding

6 Tauri-команд: `auth_start_login`, `auth_get_access_token`, `auth_is_authenticated`, `auth_current_email`, `auth_logout`, `gmail_inbox_count`, `gmail_random_message`

### Фронтенд (Vue 3 + Quasar)

Одна сторінка (`Login.vue`) з трьома режимами:

1. **Незалогіненийе:** кнопка «Увійти через Google»
2. **Завантаження:** skeleton-анімація поки підтягуються дані
3. **Залогінений:** показує email, кількість листів і випадковий лист (From / Subject / Date / Body до 10к символів) + кнопки «Показати інший» і «Вийти»

Все — українською, з обробкою кожного типу помилки через `i18n/auth-errors.js`.
