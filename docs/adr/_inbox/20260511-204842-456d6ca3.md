## ADR Ініціалізація C4-документації MLMaiL у `docs/ci4/`

**Контекст:** Проєкт MLMaiL (Tauri 2 + Vue 3, цілі платформи macOS і Android) не мав жодної архітектурної документації, хоча правило `.cursor/rules/n-ci4.mdc` вимагає, щоб C4-діаграми існували поряд із кодом і були джерелом істини перед будь-якою зміною.

**Рішення/Процедура/Факт:** Створено канонічне розташування `docs/ci4/` із шістьма файлами: `README.md` (навігація і статус), `01-context.md` (System Context: користувач, Google Identity Services, Gmail API, LLM/TTS-провайдери), `02-containers.md` (MLMaiL Frontend — Vue 3/Vite, MLMaiL Backend — Rust/Tauri 2, WebView, локальне сховище; два варіанти розгортання — macOS і Android), `03-components.md` (компоненти Vue: Auth, Inbox, Reader, ActionBar, Drafter + сервіси GmailService, SummaryService, SpeechService, NotesBridge; компоненти Rust: Backend Entry, Notes Commands, Opener, Capabilities), `04-code.md` (ключові файли і сигнатури з прив'язкою до `app/src/` і `app/src-tauri/src/`), `decisions.md` (5 прийнятих рішень і 6 відкритих питань, що потребують ADR). Після генерації виправлено технічні транслітерації у `.cspell.json` (додано слова `саммері`, `саммаризація`, `PKCE`, `фронтенд`, `рантайм` та ін.) і виправлено два друкарські помилки в `.md`-файлах. Фінальні перевірки: `markdownlint-cli2` — 0 errors, `cspell` — 0 issues, `jscpd` — 0 clones.

**Обґрунтування:** Правило `n-ci4.mdc` встановлює «Rebuild Test»: якщо видалити `src/` і дати агенту лише `.md`-файли — він має відновити кодову базу. Тому C4 писалась на основі реального коду (`lib.rs`, `tauri.conf.json`, `vite.config.js`, `package.json`), а не абстрактно. Документація чесно розрізняє поточний стан (стартовий каркас з командою `greet`) і `planned`-компоненти (Gmail-інтеграція, LLM, TTS), щоб уникнути дезінформації агентів. Блоки «Тести» у кожному компоненті явно позначені як прогалина, бо тестів ще немає.

**Розглянуті альтернативи:** Не обговорювались — розташування `docs/ci4/` і структура файлів `01–04 + README + decisions` є канонічними за правилом `n-ci4.mdc`.

**Зачіпає:** `docs/ci4/README.md`, `docs/ci4/01-context.md`, `docs/ci4/02-containers.md`, `docs/ci4/03-components.md`, `docs/ci4/04-code.md`, `docs/ci4/decisions.md`, `.cspell.json` (додано словниковий запис з 15+ термінів).
