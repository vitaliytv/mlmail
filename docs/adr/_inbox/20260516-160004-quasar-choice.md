## UI-фреймворк: Quasar 2 з macOS material look

**Контекст:** MLMaiL — Tauri-застосунок для macOS та Android. Потрібен Vue
UI-кит що дає: Material Design (рідний Android UX), готові кнопки/картки/
скелетони з доступністю, cross-platform dark mode, прийнятний bundle для
desktop-бінарника.

**Рішення:** Quasar 2 з sass-vars overrides (`$primary: #0a84ff`,
`$button-border-radius: 6px`, system-ui font stack) для macOS-friendly look.
`iconSet: 'material-symbols-outlined'`, `config.dark: 'auto'`.

**Альтернативи:**

- Plain HTML + scoped CSS — нуль deps, але скелетони, loading, dialog,
  toast, ripple, safe-area — все вручну.
- Reka UI (headless Radix Vue) + Tailwind — максимальний контроль над
  виглядом, але немає готових card/chip/spinner; потрібна власна design-system.

**Чому Quasar:** bundle-size прийнятний для Tauri (10 MB binary); Android
Material = рідний UX; q-pull-to-refresh, q-virtual-scroll, $q.notify,
$q.dialog — знадобляться в наступних ітераціях MLMaiL і є з коробки.
macOS look виправляється 6 рядками sass-vars без рефакторингу компонентів.
